1147. Longest Chunked Palindrome Decomposition

Dec 23, 2019 | leetcode | 阅读

题目大意

给定一个字符串s，对其进行切分，切分后一共有k的段。要求最大的k，使得以段为一个单位来看，能够形成一个回文段。例如text = "ghiabcdefhelloadamhelloabcdefghi"，切分为7段，"(ghi)(abcdef)(hello)(adam)(hello)(abcdef)(ghi)"，把段看成一个单位，形成一个回文段。 <!-- more -->

解题思路

贪心法递归，当子串s[0:k]与s[len - k: len]相等时，递归求子串s[k:len-k]能够形成回文段的最大段数，将这个段数加2，既为整个串的最大段数。
